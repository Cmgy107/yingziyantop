# 初遇籽言 · 第一年星光抽卡小程序

一个纯静态网页抽卡小程序，可直接通过 GitHub Pages、阿里云 OSS、宝塔面板或任意静态网站服务发布。

## 功能

- 单抽、十连抽
- 十连 95 折，十连保底 SR 或以上
- 100 抽 UR 保底
- 抽到卡后先展示卡背，点击翻开
- 卡池详情、抽卡记录、卡牌仓库
- 重复卡回收为种籽
- 3 张重复卡兑换同级任意其他卡
- 兑换码领取种籽

## 本地打开

直接双击 `index.html` 即可打开。

如果浏览器限制本地资源，也可以在本目录运行：

```bash
python -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173/
```

## 数据说明

抽卡记录、种籽余额、卡牌仓库存储在浏览器本地 `localStorage` 中。不同设备、不同浏览器之间默认不会同步。
